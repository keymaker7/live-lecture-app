# 슬라이드 연수 (Live Lecture)

실시간 슬라이드 연수 플랫폼 — Vercel 배포용 Next.js 앱

**배포 주소:** https://live-lecture-n1yt3352b-jkeymaker-5915s-projects.vercel.app

## 로컬 실행

```bash
npm install --legacy-peer-deps
npm run dev
```

→ http://localhost:3030

## Supabase 설정

`.env.local` 파일:

```env
NEXT_PUBLIC_SUPABASE_URL=https://xxxxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=sb_publishable_...
```
